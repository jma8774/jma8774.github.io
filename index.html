My Academic Projects


This is a food delivery app called Foodies developed by a group of friends for our Software Engineering class. There was a specific set of criteria that each project must fulfill and for me, my job was to create a working map using Google Map API, implementing a voice recognition feature, a food bidding system for the delivery person, as well as quality insurance to make sure everyone knows what they're doing because it is their first time using Python and working with GUI. We had to do it in a very ugly way.

The first thing I worked on doing is learning how to use the Google Map API to retrieve relevant information to display, such as the distance from one location to another, the time it would take, etc. Once the delivery person completes their delivery, they can rate the customer (one of the criteria) and then the system will update the information.

The second thing I worked on is the bidding system (another criteria), where delivery people will have to compete with others to bid on an order to deliver. I think this is a very dumb thing that an app should have but it was required by our professor. I have a timer running down and whenever a new order was placed, it would appear in the list. Each bidding lasts a minute.

The last thing I had to work on was making a voice recognition feature (another criteria), which allows handicapped people to place food orders when using our app. I made a very basic version using the SpeechRecognition and PyAudio modules.


After combining everyone's work together and fixing my team's bugs, we presented our project to our professor and received a grade of 92 which was acceptable because some of us are inexperienced with Python.
